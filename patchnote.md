# V1.1

## Features
- Script de désinstallation `install/remove.sh`.
- Option -U pour mettre à jour automatiquement (experimental).
- Option -v, --version, pour férifier la version locale actuelle.
- Scripts de test dans un environement docker `./test`.
- Nettoyage des temlplates et commandes obsolètes (lire `lib/laravel/readme.md`).
- `.env` ajouté dans les fichiers de configuration de l'utilisateur à la racine.
